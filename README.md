## Description

選択したアートボードからcssの記法でフォント情報を全て抜き出し、
クリップボードに貼り付けるツールです。(公開しているプラグインではないため下記のUsageは将来公開できたとき用のものです。)

## Usage

1. アートボードを選択した状態でプラグインの「Get Text」を選択。
2. 任意のファイルなどに`ctrl + v`でペースとしておく
